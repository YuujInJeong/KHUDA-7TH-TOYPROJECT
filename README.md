# 🇰🇷 **나랏말싸미: 한글 맞춤법 학습 서비스**

## **프로젝트 소개**

---

나랏말싸미는 사람들이 자주 틀리는 맞춤법 문제를 바탕으로 사용자의 한국어 맞춤법 실력을 파악하고, 추천 시스템을 활용해 적절한 문제를 제공하는 학습 서비스입니다. 단계별 문제 풀이를 통해 사용자는 자연스럽게 맞춤법 실력을 향상시킬 수 있습니다.
